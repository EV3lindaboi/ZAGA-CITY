What is Zaga?

Zaga is a map that is aimed to be on most big shooters with a Deathmatch game mode.

Want to help?

You can! Help is always welcome. Grab the Doom 2 version of Zaga to get started. Feel free to make as many modifications as possible, as long as the layout remains close to the original.

The Zaga City Github repo

The Zaga City Github repo is where all the official and community made variants of Zaga will be stored. Click here to access it.
